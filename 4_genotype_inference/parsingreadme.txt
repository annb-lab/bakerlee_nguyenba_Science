10xmer 13-locus genotype data parsed in terminal

1. open a partition
2. load anaconda module
3. conda install python-levenshtein to the environment
4. then call something like "python CRISPR_parse_10xmer-loci_v01.py <csv with file names> <destination folder> <fastq files> 0 CRISPR_10xmer-g"

make sure milo_tools and all the files being called are in the appropriate directory first.
